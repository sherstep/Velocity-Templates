## Use Template 1 as a day-0 or day-n template to prepopulate description and access/trunk for new switch ##
## Use Template 2 as a day-n template to push configs based on description ##
## Can use interface range command, but be aware of 8 range limitation per command ##
## Can also use for-each loop for a long range consecutive ports ##

TEMPLATE #1 

interface range gi1/0/1-10,13,15,20-22
description Camera Interfaces
switchport mode access

interface range gi1/0/11-12,14,16-19
description Access Point Interfaces
switchport mode access

interface range TenGig1/1/3-4,Twenty1/1/1-2
description Uplink Ports
switchport mode trunk
